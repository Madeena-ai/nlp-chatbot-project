## 📌 About

## 🛠️ Tech Stack
- Python 3.8+
- NLTK
- Scikit-learn
- TF-IDF Vectorization
- Naive Bayes Classifier
- Pandas
- Flask / Streamlit

## 🔄 NLP Pipeline
User Input → Preprocessing → TF-IDF → Naive Bayes → Response
